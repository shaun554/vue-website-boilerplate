# Config
>To run the application go to .env and give bucket credentials. also provide
>MAILGUN_KEY :'',
>MAILGUN_DOMAIN : ''
## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:3000
npm run dev

# build for production with minification
npm run build

# build for production
npm run start
```
